### How to configure project
`uv pip install -e ".[dev]"`  
`pre-commit install`

### How to run research
- Run LMStudio
- Put articles of interest in the “to research” folder.
- Run `python3 rag.py`

### How to get results
- Run `python3 -m http.server 8000`
- Open `localhost:8000` in browser
